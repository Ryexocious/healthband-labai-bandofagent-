# HealthBand Design Context

## Product
Multi-agent medical advisory dashboard for the Band (Thenvoi) hackathon. Users submit symptoms, IoT vitals, and files; five AI agents analyze and produce a health assessment.

## Audience
Demo operators, hackathon judges, technical clinicians evaluating agent orchestration. State of mind: focused, evaluating system credibility.

## Color Strategy
**Committed** SaaS palette inspired by modern CRM dashboards: soft blue-lavender canvas, white cards, indigo-to-cyan gradient brand, per-agent color avatars.

## Theme
Bright clinical workspace like Sugar CRM / Supalab: airy, colorful, engaging. Floating gradient orbs, stat chips, colored scenario cards, gradient CTA.

## Typography
- Display/UI: Plus Jakarta Sans
- Data/channels: IBM Plex Mono with tabular nums

## Shape
- Shell cards: 20px outer radius, 16px inner
- Buttons: full pill
- Inputs: 12px radius

## Motion
- UI interactions: 160-220ms, cubic-bezier(0.23, 1, 0.32, 1)
- Enter stagger: 40ms between items
- No animation on repeated actions (tab switches)
- prefers-reduced-motion: opacity only

## Anti-goals
Generic AI purple gradients, Inter font, emoji icons, equal three-card feature rows, glassmorphism everywhere without purpose.
