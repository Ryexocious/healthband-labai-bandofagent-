# Product
## Register
brand
## Users
Primary users are clinicians, care coordinators, and operations teams reviewing AI-assisted triage output for fast decision support. Secondary users are technical stakeholders demoing the multi-agent workflow to partners and evaluators.
## Product Purpose
HealthBand turns symptom, vitals, and attachment input into a coordinated multi-agent medical advisory report with transparent channel-level collaboration. Success means users can submit a case quickly, monitor agent reasoning clearly, and receive an actionable specialist-ready summary without ambiguity.
## Brand Personality
Clinical, decisive, and premium. The interface should feel trustworthy and high-signal, with clear hierarchy and confident visual polish inspired by modern SaaS control-room aesthetics.
## Anti-references
Avoid template-looking repeated cards, avoid neon/gamified visual noise, and avoid ultra-minimal sparse layouts with weak hierarchy. Avoid generic dashboard tropes that make the UI feel interchangeable.
## Design Principles
1. Prioritize signal over decoration: every visual element should improve comprehension or decision speed.
2. Make orchestration visible: agent progress and channel communication must read as a coherent narrative.
3. Commit to a distinct visual voice: premium, contemporary, and non-generic without sacrificing readability.
4. Preserve operational trust: emergency and critical states must remain immediate, unmistakable, and accessible.
## Accessibility & Inclusion
No strict formal conformance target was specified for this redesign. Maintain practical usability standards: readable contrast for core content, clear focus states, and reduced cognitive load in dense information areas.
